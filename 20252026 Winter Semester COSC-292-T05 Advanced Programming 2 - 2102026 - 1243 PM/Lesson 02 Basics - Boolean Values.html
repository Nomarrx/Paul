Lesson 02 Basics - Boolean Values

Boolean Values

Quick highlights:

  • Legacy: 0=false, non-zero=true
  • Use <stdbool.h>: bool, true, false

Details

C originally had no Boolean data type. Boolean comparisons were done with any value (including char and float and double) where 0 was false and any non-zero value was true. Such comparisons can still be used.

In C99, a _Bool type was introduced. This type still stores integers, but only 0 and 1. Assigning any non-zero value to a _Bool type is allowed, but it will be stored/displayed as 1.

There is also a system header file called <stdbool.h> which contains a #define directive that redefine bool as _Bool (so you can use bool as a data type if you include this header – it will be the same as _Bool). It also contains #define directives to redefine true as 1 and false as 0, so you can use true or false in place of 1 and 0 (but the values are still the integers 1 and 0 in the underlying code).

If you want to see what stdbool.h looks like (or any other system header file), go to C:\Program Files\Microsoft Visual Studio\2022 and search for the filename. For instance, on my system, stdbool.h is at C:\Program Files\Microsoft Visual Studio\2022\Enterprise\VC\Tools\MSVC\14.34.31933\include and contains the following code:

//
// stdbool.h
//
// Copyright (c) Microsoft Corporation. All rights reserved.
//
// The C Standard Library <stdbool.h> header.
//

#ifndef _STDBOOL

#define _STDBOOL

#define __bool_true_false_are_defined 1

#ifndef __cplusplus

#define bool _Bool

#define false 0

#define true 1

#endif /* __cplusplus */

#endif /* _STDBOOL */

Note that all stdbool.h is doing is redefining bool, true, and false so that they will be replaced with their corresponding values when the pre-processor finds them in the code. (We will learn more about the structure of header files later.)

Let’s try out Boolean values, both as integers and using the _Bool type. First, go to the top of the file and add the include directive for stdbool.h:


#include <stdbool.h>

Then try adding the following code in main:

booleanDemo();

Then above main, add the following booleanDemo function:


void booleanDemo()
{

int myValue = 3;

// boolean values are integers - 0 for false, 1 for true


	printf("False comparison prints as: %d\n", (myValue > 3));
	printf("True comparison prints as: %d\n", (myValue <= 3));

if (myValue) // any non-zero integer is true


	{
		printf("3 is non-zero, so it is considered true\n");
	}

// In C99 and up, there is a Boolean type called _Bool.

// It still stores integer values (either 0 or 1).

_Bool myBool = 3; // any non-zero integer is still true, prints as 1


	printf("Boolean myBool: %d\n", myBool);

// We can use the <stdbool.h> header file to define

// bool (which is just _Bool redefined),

// true (which is really 1), and false (which is really 0)

bool myRedefinedBool = false;


	printf("Boolean myRedefinedBool: %d\n", myRedefinedBool);
}

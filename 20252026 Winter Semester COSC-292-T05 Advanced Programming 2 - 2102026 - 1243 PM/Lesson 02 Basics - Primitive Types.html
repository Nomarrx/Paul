Lesson 02 Basics - Primitive Types

Primitive Types

Quick highlights:

  • char, int, float, double
  • Use sizeof to confirm sizes

Details

C provides four primitive types: char, int, float, double.


char is used for small integer values. It is required to be at least 8 bits or 1 byte in length (but could be more on some systems – on our systems, it is 8 bits or 1 byte). Note that char variables can contain numbers or characters. When used as a character, it can store ASCII codes (8-bit) but not Unicode values that exceed its size.

int is used for larger integer values. It is required to be at least 16 bits or 2 bytes in length (but could be more on some systems – on our systems, it is 32 bits or 4 bytes).

float is used for single-precision floating-point values.

double is used for double-precision floating-point values.

Let’s try writing a program in a new project in a source file named something like program.c to illustrate these types. Start with some include directives and one define directive:


#include <stdio.h>
#include <stdlib.h>

#define PI 3.141519f // f at the end of the literal denotes it as a float literal

Then create the main function. Note that we will use the printf function, which is included in <stdio.h>, for output. printf is similar to the Java method System.out.printf() – in fact, the Java method was based on the C function. printf displays a message to the standard output stream, which by default is the console.

For more information about any C function and in particular its Visual Studio implementation, google the function name and Microsoft. For instance, searching for printf microsoft will allow you to find https://learn.microsoft.com/en-us/cpp/c-runtime-library/reference/printf-printf-l-wprintf-wprintf-l?view=msvc-170

The main function will look like this:

// Program starts with the main function


int main(int argc, char* argv[])
{

// Use printf to output to the console


	printf("Hello"); // does not automatically move to a new line
	printf(" world\n\n"); // \n specifies a newline

primitiveTypes();

return EXIT_SUCCESS; // defined in stdlib.h


}

Note the call to the primitiveTypes function – before we run this program, we need to create it. The C compiler needs to know what primitiveTypes looks like before it calls it in main, so put it above the main function; otherwise, you will get an error that primitiveTypes has been redefined (in other words, the call in main assumes that primitiveTypes is different from how the function is coded). So above main, write:


void primitiveTypes()
{
	char myCharacter = 'A'; // character literals are in single quotes

int myValue = 4276803;

float myDigits = PI; // using PI from the #define above

double myLongerDigits = 12345.6579;

// Print the value of the variables using format specifiers that start with %


	printf("The value of c is %c\n", myCharacter);
	printf("The value of c as an int is %d\n", myCharacter);
	printf("The value of x is %d\n", myValue);
	printf("The value of x as a character is %c\n", myValue);
	printf("The value of x in hex is %x\n", myValue);

// Can use the format specifier to modify output

// For instance, to display 10 characters with 2 decimals


	printf("The value of d is %10.2f\n", myLongerDigits);
}

Note that character literals are expressed in single quotes like 'A', so we can assign that to a char. We then use printf to print the values in different formats. We can print the char variable as a character using the %c format specifier or as a number using %d. We can also print the int value as a number, as a character (using only the low-order 8 bits), or in hexadecimal using %x. We can print the double in various fashions, including modifying the number of digits it prints (try printing it without the 10.2 in the format specifier to see the difference). More about printf format specifiers in Visual Studio can be found at https://learn.microsoft.com/en-us/cpp/c-runtime-library/format-specification-syntax-printf-and-wprintf-functions?view=msvc-170

How big are these primitive types and thus what sizes of values can they store? In C, there are certain guarantees – for instance, a char must be at least 8 bits (1 byte), and an int must be at least 16 bits (2 bytes). But the exact sizes depend on the environment and operating system used. There is an operator called sizeof in C that will tell you the size of a variable or data type on the system you are using – see https://learn.microsoft.com/en-us/cpp/c-language/sizeof-operator-c?view=msvc-170. So let’s add the following code to primitiveTypes – note the use of the format specifier %zu for the results of sizeof:

// What are the sizes of each data type on our system? (may vary with system)

// The sizeof operator gives the size of a data type, struct, array, etc.

// Measured in bytes


	printf("The size of a char is %zu\n", sizeof myCharacter);
	printf("The size of an int is %zu\n", sizeof myValue);
	printf("The size of a float is %zu\n", sizeof (float));
	printf("The size of a double is %zu\n", sizeof (double));

You will find that on our systems that char is 1 byte (8 bits) in size; int (which is only guaranteed to be 16 bits or 2 bytes) is 4 bytes in size; float is 4 bytes; and double is 8 bytes.

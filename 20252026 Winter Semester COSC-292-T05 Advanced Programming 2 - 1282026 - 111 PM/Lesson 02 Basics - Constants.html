Lesson 02 Basics - Constants

Constants

Quick highlights:

  • Use const to disallow reassignment
  • Literal suffixes: e.g., 3.14f

Details

C can also have constants. Use the keyword const in front of the variable declaration and set the variable equal to its value on the same line as the declaration. A constant cannot be assigned a new value.

Try adding the following code in main:

constantsDemo();

Then above main, add the following constantsDemo function:


void constantsDemo()
{

// Note keyword const

const float pi = 3.141519f;

//pi = pi + 1; // cannot change the value – see error if uncommented


	printf("The value of pi is %f\n", pi);
}

In this class, we will be covering a procedural programming language, C. C is a popular language with low-level capabilities. C++ is an object-oriented superset of C.

Object-oriented vs. Procedural

  • OO: decide what classes you need, provide a full set of operations for the class, and make commonality explicit using inheritance; separate public/private
  • Procedural: decide which procedures and data structures you want and use best algorithm; support is based on functions communicating by passing arguments and returning values

Creating a C Program

  • Produce a source file (.c or .cpp)
  • Compile to object code (.o or .obj)
  • Link the object file with C libraries to produce an executable program (.exe)

Steps:

  • Start Visual Studio 2022
  • Create a new project
  • Choose C++ from the list of languages and choose an Empty C++ Project
  • Create the project in C:\VisualStudioProjects in SaskPolytech to deal with security issues
  • From the project, go to Source Files and right-click | Add New Item... and type in the file name with the extension .c
  • Create main function, add libraries, create other functions
  • Compile and run using Ctrl+F5

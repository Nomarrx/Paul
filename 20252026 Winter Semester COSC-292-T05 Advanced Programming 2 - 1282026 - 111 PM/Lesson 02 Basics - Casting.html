Lesson 02 Basics - Casting

Casting

Quick highlights:

  • Implicit casts may truncate
  • Explicit cast: (type)expr

Details

C does implicit casting whenever it can. Unlike Java, it will implicitly cast a larger data type to a smaller data type without error (Visual Studio will show a warning), even if it results in loss of precision. You can also explicitly cast values by using the data type in parentheses (as in Java).

Try adding the following code in main:

castDemo();

Then above main, add the following castDemo function:


void castDemo()
{

int myInt;

float myFloat = 3.2; // 3.2 is a double literal, implicitly cast to float

float fahr = 115; // 115 is an integer literal, implicitly cast to float

float celc; // Note that uninitialized values may have random numbers

// Assign myFloat to myInt

myInt = myFloat; // implicitly cast


	printf("The value of myInt is %d\n", myInt);

myInt = (int)myFloat; // explicitly cast


	printf("The value of myInt is %d\n", myInt);

// Temperature conversion


	printf("The temperature in Fahrenheit is %f\n", fahr);

celc = ((float)5 / 9) * (fahr - 32); // cast to avoid integer division

// could also use 5.0 or 5.0f, 9.0 or 9.0f, etc.


	printf("The temperature in Celsius is %f\n", celc);
}
